# 1819_IHDCB132_Jacquet_Samuel
####### Jumping Banana #######

## Compiling with "make" command.
## Running with "./prog" on Linux or "prog" on Windows.

## Game keyboard. Arrow keys.

* Home : Go left.
* End : Go right.
* PgUp : Jump actived.
* PgDn : climb to the down.

## Menu

* PgUp 	: Monter dans le menu
* PgDn 	: Descendre dans le menu
* Enter : Entrer
* ESC 	: Sortir
